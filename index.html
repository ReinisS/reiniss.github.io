PyPrice - A tool for tracking price changes on Amazon.com items PyPrice - Rīks priekš cenu izmaiņu izsekošanas Amazon.com internetveikala precēm

PyPrice lets you add links to Amazon.com items and track their prices overtime. At the click of a single button PyPrice automatically visits each of the previously added Amazon.com pages, loads the contents of the website and finds the price of each given item. Afterwards, the price is saved in a JSON file locally. The gathered datapoints can then be viewed in easy-to-understand graphs. PyPrice ļauj pievienot saites uz Amazon.com precēm un sekot līdzi to cenu izmaiņām ilgākā laika periodā. Ar vienu pogas klikšķi PyPrice automātiski iziet cauri katrai iepriekš pievienotajai Amazon.com lapai, ielādē tās saturu un atrod katras dotās preces cenu. Pēc tam šī cena tiek saglabāta lokālā JSON failā. Ievāktos preču cenu datus pēc tam var apskatīt viegli uzskatāmās diagrammās.

Used technologies: Izmantotās tehnoloģijas:

  • Python
  • PyQt5
  • Scrapy
  • Matplotlib

OperatorFinder - A console program for finding missing '+' and '-' operators between numbers OperatorFinder - Konsoles programma priekš '+' un '-' operatoru atrašanas starp skaitļiem

This console program finds all of the possible combinations of '+' and '-' operators which can be used in-between the given lists of numbers to reach the given targets. The input numbers are read from a text file.
Example input: 17 3 2 5 = 23
Output: 17 + 3 - 2 + 5 = 23 Šī konsoles programma atrod visas iespējamās '+' un '-' operatoru kombinācijas, kuras var tikt ievietotas starp padotajiem skaitļiem, lai iegūtu padoto rezultātu. Ieejas skaitļi tiek nolasīti no teksta (.txt) faila.
Ieejas piemērs: 17 3 2 5 = 23
Iegūtais rezultāts: 17 + 3 - 2 + 5 = 23
